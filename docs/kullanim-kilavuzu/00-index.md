## Ekran görüntüsü çekme rehberi

**macOS**: `Cmd + Shift + 4` → alan seç (Cmd + Shift + 5 ile pencere de çekebilirsin).
**Windows**: `Win + Shift + S` (Snip & Sketch) → alan seç → kopyala/kaydet.

**Boyut önerisi:**
- Tam sayfa: 1280×720 veya 1440×900
- Tek kart/buton: 600×400 yeterli
- Mobil görünüm: 375×812 (iPhone SE)

**İsim kuralı (önerilir):**
```
gorseller/01-giris-ekrani.png
gorseller/02-anasayfa-kpi-kartlari.png
gorseller/03-yeni-ogrenci-formu.png
```

**Hassas bilgi maskeleme:**
- Gerçek öğrenci/veli adı, telefon, TC: **bulanıklaştır** (Preview > Annotate > Blur)
- Demo verisi kullan (Ali Demo, Ayşe Test)
- Yedek dosya isimleri ve URL'lerde tenant slug görünebilir — sorun değil
